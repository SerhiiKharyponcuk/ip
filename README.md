# СЛЕД://ТЫ

Статический демонстрационный сайт в киберпанк-стиле. Показывает публичный
IP-адрес, примерный город и страну по IP, а после явного разрешения пользователя —
точные координаты через браузерный API геолокации.

## Локальный запуск

Используйте любой локальный HTTP-сервер, например:

```bash
python3 -m http.server 8080
```

Затем откройте `http://localhost:8080`.

## Публикация на GitHub Pages

1. Загрузите файлы в публичный репозиторий GitHub.
2. Откройте **Настройки → Страницы**.
3. В разделе источника публикации выберите **GitHub Actions**.
4. Готовый процесс из `.github/workflows/pages.yml` опубликует сайт автоматически.

Точная геолокация в обычном браузере работает только через HTTPS или на
`localhost`. GitHub Pages автоматически предоставляет HTTPS.

## Конфиденциальность

- Данные не записываются в базу и не отправляются владельцу сайта.
- IP-данные загружаются непосредственно в браузере с `ipwho.is`.
- Координаты остаются только во вкладке пользователя до её закрытия.
- Доступ к камере не запрашивается; сообщение о камере — только шутка.
