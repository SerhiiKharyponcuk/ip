# TRACE://YOU

Статичний демонстраційний сайт у кіберпанк-стилі. Показує публічну IP-адресу,
приблизне місто/країну за IP та, лише після явного дозволу користувача, точні
координати через браузерний Geolocation API.

## Локальний запуск

Через будь-який локальний HTTP-сервер, наприклад:

```bash
python3 -m http.server 8080
```

Потім відкрийте `http://localhost:8080`.

## Публікація на GitHub Pages

1. Завантажте файли в публічний GitHub-репозиторій.
2. Відкрийте **Settings → Pages**.
3. У **Build and deployment** виберіть **Deploy from a branch**.
4. Виберіть гілку `main`, папку `/ (root)` і натисніть **Save**.

Точна геолокація у звичайному браузері працює тільки через HTTPS або на
`localhost`. GitHub Pages автоматично надає HTTPS.

## Приватність

- Дані не записуються у базу та не надсилаються власнику сайту.
- IP-дані завантажуються безпосередньо у браузері з `ipwho.is`.
- Координати залишаються тільки у вкладці користувача до її закриття.
- Доступ до камери не запитується; повідомлення про камеру — лише жарт.
